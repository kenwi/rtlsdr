# rtlsdr

Files from my work with rtl-sdr. Adding nodes and functionality as I go.

# Requirements
* http://kmkeen.com/rtl-power and whatever dependencies.
* http://kmkeen.com/tmp/heatmap.py.txt
* A rtl-sdr dongle. Go buy one.
